# PRD 초안 작성 자동화 — LLM 업무 자동화 과제

LLM을 활용해 **PRD(제품 요구사항 정의서) 초안 작성**을 자동화하는 프롬프트 패키지입니다.
동일한 시스템 프롬프트를 3종 LLM에 입력해 비교·선정하고, 시스템을 설계하고, 실제 대화로 검증한 결과물을 담았습니다.

## 과업 개요

- **과업:** PRD 초안 작성 (예시 기능: 바코드 스캔 재고 등록)
- **타겟 사용자:** 사내 PM·기획자
- **페르소나:** 'PRD 도우미 진' (10년차 시니어 프로덕트 매니저)
- **최우선 기준:** 규칙 준수 — 확인 안 된 정보를 임의로 창작하지 않음
